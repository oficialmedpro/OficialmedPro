🎯 Teste - Daily Performance Table

📊 O que foi implementado:

✅ Componente DailyPerformanceTable.jsx - Tabela com estrutura idêntica à imagem
✅ CSS DailyPerformanceTable.css - Estilo personalizado com cores corretas
✅ Service dailyPerformanceService.js - Busca dados de leads/oportunidades por dia
✅ Integração no DashboardPage.jsx - Aparece após chart-section quando funil selecionado

🧪 Como testar:

  1. Execute o projeto: npm run dev
  2. Acesse a dashboard no navegador
  3. Selecione um funil específico (não "all")
  4. Selecione uma unidade específica (não "all")
  5. A tabela "Performance Diária" deve aparecer após o chart-section
  6. Verifique se as cores estão corretas (fundo escuro, texto branco)
  7. Verifique se os gaps negativos aparecem em vermelho
  8. Verifique se a tabela está alinhada corretamente

🔧 Dados implementados:

✅ Leads: Busca oportunidades criadas por dia do Supabase
✅ Faturamento: Soma dos valores das oportunidades por dia
✅ Ticket Médio: Faturamento dividido por número de leads
⏳ N° Vendas: Aguardando onde buscar os dados
⏳ Taxa Conversão: Aguardando como calcular

🎨 Estrutura visual:

  1. Header: "Performance Diária - [mês/ano]"
  2. Colunas: Leads, N° Vendas, Faturamento, Taxa Conversão, Ticket Médio
  3. Sub-colunas: Realizado, Meta, Gap
  4. Linhas: Resumo do mês + dias do mês (1º até hoje)
  5. Cores: Fundo escuro, texto branco, gaps negativos em vermelho
